# Sprint Planning Checklist

> **Purpose**: Ensure proper test planning during sprint planning.

---

## Story Review

- [ ] All stories reviewed for testability
- [ ] Acceptance criteria verified
- [ ] Test effort estimated
- [ ] Dependencies identified

---

## Test Planning

- [ ] Test strategy for each story
- [ ] Automation candidates identified
- [ ] Manual test needs noted
- [ ] Environment requirements listed

---

## Capacity Planning

- [ ] Test automation capacity allocated
- [ ] Manual testing effort included
- [ ] Bug buffer considered
- [ ] Technical debt time allocated

---

## Risk Assessment

- [ ] High-risk features identified
- [ ] Regression test scope defined
- [ ] NFR testing needs planned
- [ ] Integration test needs identified
